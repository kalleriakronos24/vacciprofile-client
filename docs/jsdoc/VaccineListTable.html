VaccineListTable

  • VaccineListTable Component

Source:
  • components/information/VaccineListTable.js, line 3
Example
// Render the VaccineListTable component with dummy data and functions
<VaccineListTable 
  activeTab="Vaccine"
  selectedPathogen={{ name: 'Pathogen X' }}
  selectedVaccine={{ name: 'Vaccine Y' }}
  selectedLicenser="Licenser Z"
  handleSelectPathogen={pathogen => console.log('Pathogen selected:', pathogen)}
  handleSelectVaccine={vaccine => console.log('Vaccine selected:', vaccine)}
  handleSelectLicenser={licenser => console.log('Licenser selected:', licenser)}
  getVaccinesByManufacturer={() => [{ name: 'Vaccine Y', licenser: ['Licenser Z'] }]}
  getPathogenByVaccine={vaccine => ({ name: 'Pathogen X' })}
/>
Parameters:
Name Type Description
props Object

The component accepts activeTab, selectedPathogen, selectedVaccine, selectedLicenser, and several handler and data functions as props.

Properties
Name Type Description
activeTab string

The type of detail currently selected, e.g., "Vaccine", "Pathogen", or "Licenser".
selectedPathogen Object

The currently selected pathogen object.
selectedVaccine Object

The currently selected vaccine object.
selectedLicenser string

The currently selected licenser.
handleSelectPathogen function

Function that gets triggered when a pathogen is selected.
handleSelectVaccine function

Function that gets triggered when a vaccine is selected.
handleSelectLicenser function

Function that gets triggered when an licenser is selected.
getVaccinesByManufacturer function

Function that returns a list of vaccines based on the manufacturer.
getPathogenByVaccine function

Function that returns the pathogen associated with a specific vaccine.
Returns:

The Manufacturer Information Table component.
