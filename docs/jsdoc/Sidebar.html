Sidebar

Namespace

Sidebar

Description:

  • Sidebar Component

    A component that displays a sidebar for selecting manufacturers, products, or pathogens. It supports changing the active tab and handling item selection.

Source:
  • components/Sidebar.js, line 3
Example
// Example usage of Sidebar component
<Sidebar 
   activeTab="Manufacturer"
   setActiveTab={(type) => console.log(type)}
   sidebarList={[{ name: 'ItemA' }, { name: 'ItemB' }]}
   selectedManufacturer={{ name: 'ItemA' }}
   selectedVaccine={{ name: 'ItemB' }}
   selectedPathogen={{ name: 'ItemC' }}
   selectedLicenser={{ name: 'ItemC' }}
   setSelectedManufacturer={(item) => console.log(item)}
   setSelectedVaccine={(item) => console.log(item)}
   setSelectedPathogen={(item) => console.log(item)}
   setSelectedLicenser={(item) => console.log(item)}
   changedFrom='Sidebar'
   setChangedFrom={(source) => console.log(source)}
   italizeScientificNames={text => text.replace(/(SARS-CoV-2)/g, '<i>$1</i>')}
/>
Parameters:
Name Type Description
props Object

The component accepts various props to handle sidebar functionality.

Properties
Name Type Description
activeTab string

The currently active tab, which can be 'Manufacturer', 'Product', or 'Pathogen'.
setActiveTab function

Function to set the type of details to be displayed ('Manufacturer', 'Product', or 'Pathogen').
sidebarList Array

List of items (manufacturers, products, or pathogens) available for selection.
selectedVaccine Object

The currently selected product (vaccine).
selectedPathogen Object

The currently selected pathogen.
selectedManufacturer Object

The currently selected manufacturer.
selectedLicenser Object

The currently selected licenser.
setSelectedVaccine function

Function to update the selected product (vaccine).
setSelectedPathogen function

Function to update the selected pathogen.
setSelectedManufacturer function

Function to update the selected manufacturer.
setSelectedLicenser function

Function to update the selected licenser.
changedFrom String

The param where the selected item change took place
setChangedFrom function

Function to set the source of the change triggering the main update.
italizeScientificNames function

Function that converts scientific names in the description to italicized text.
Returns:

The Sidebar component for selecting items and updating the main based on the active tab.

Type
JSX.Element

Methods

(inner) handleClickSidebar(item)

Description:

  • Handles the click events for sidebar items based on the active tab. It selects or unselects an item depending on the current selection state. The behavior varies depending on whether the item belongs to the Manufacturer, Product, or Pathogen tab.
